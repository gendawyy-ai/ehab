# Ehab Sherif - Senior Property Consultant Website

A premium real estate portfolio website built with Flask (Python), HTML5, CSS3, and JavaScript.

## Features
- **Dynamic Content**: All consultant info and project details are loaded from `data.json`.
- **Premium Design**: Modern aesthetics with animations, navy & gold color palette, and responsive layout.
- **Project Portfolio**: Collection of projects with monthly installment info.
- **Interactive Maps**: Each project includes a pinned Google Map location.
- **Unit Details**: Popups/Modals to view specific unit types, sizes, down payments, and installment plans.

## Project Structure
- `app.py`: Flask backend server.
- `data.json`: Centralized data store (Edit this to change phone numbers or projects).
- `templates/`: HTML templates using Jinja2.
- `static/`: CSS styles and (future) project images.

## Setup
1. `pip install flask`
2. `python app.py`
3. Visit `http://127.0.0.1:5000`
